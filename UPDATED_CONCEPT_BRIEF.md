# FindFoundFast — Updated Concept Brief

## What it is

FindFoundFast is a simple web-based micro-location guidance tool for apartment complexes, nursing homes, hospitals, offices, hotels, schools, and large residential communities.

Property managers set up each building once. Residents then get easy access to either a **Permanent link** (no expiration) or a **Timed link** (with auto-expiring gate codes) so drivers, delivery people, visitors, guests, pharmaceutical reps, or family can find the exact door without confusion.

## Resident Experience (simple & fast)

- Residents log in one time only using the 5-character code given by the property manager.
- After the first login, they can just open findfoundfast.com and immediately see their building's links.
- They can choose:
  - **Permanent link** — Never expires. Ideal when no gate code is needed or when the resident prefers to text the code separately.
  - **Timed link** — 30 minutes / 1 hour / 1 day. Used when gate codes are included (codes automatically disappear when the timer ends).
- They copy the link and paste it into DoorDash, Uber Eats, text messages, etc.

## Property Manager Control

- Sets up each building once (photos + optional gate codes).
- Decides whether gate codes are included in the building profile.
- Every resident in that building automatically benefits.

## Driver / Visitor Experience

- Clicks the link (no app, no login).
- Sees large, clear step-by-step photos.
- If it's a timed link, the gate code appears and expires with the link.
- If it's a permanent link, it stays active forever.

## Key Benefits

- One-time resident login -> super easy ongoing use
- Two link types (Permanent + Timed) give maximum flexibility
- Works for food delivery, Amazon, Uber, pharma reps, visitors, guests, medical transport, etc.
- No app required — opens in any phone browser
- Gate codes only expire when the resident wants them to
